# 🏦 Bank Management System

A simple Java console-based application that simulates basic banking operations such as creating accounts, performing deposits/withdrawals, and managing transaction records. This project is built using **object-oriented programming** concepts.

---

## 🚀 Features

- 👤 Create customer bank accounts  
- 💰 Deposit and withdraw money  
- 📋 Display account details  
- 📜 Track transaction history  
- ❌ Delete or close accounts  

---

## 🛠️ Tech Stack

- **Language**: Java  
- **Paradigm**: Object-Oriented Programming  
- **Environment**: Console-based CLI  

---

## 📂 Project Files

```
Bank_ManageMent_System/
├── Account.java         # Represents individual customer accounts
├── Bank.java            # Core logic for account management (add, delete, fetch)
├── Transactions.java    # Records and manages deposit and withdrawal logs
└── Bank_system.java     # Main application class with menu-driven interface
```

---

## 💡 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Bank_ManageMent_System.git
   cd Bank_ManageMent_System
   ```

2. Compile the Java files:

   ```bash
   javac *.java
   ```

3. Run the system:

   ```bash
   java Bank_system
   ```
